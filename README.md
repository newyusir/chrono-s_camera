# chrono's camera
an assist tool for making receipt factor picture

supported os: windows 10 build 17134 or newer

windows 10 user may need to install Webview2 runtime.

# How to use
1. open your app

1. select uma window

1. press hotkey to start capture (default is win + rshift)

1. scroll by using mouse wheel (do not scroll too fast, it might skip important frames)

1. press hotkey again to finish capture

1. app will send your picture to clipboard, paste it (click "クリップボードから貼り付け") on receipt factor site

*(note: clipboard only works on this app's browser.)*
